# Paper LaTeX Template

This folder contains the LaTeX sources for the project paper, formatted with the official IEEE Conference Paper template.

The template and formatting guidance come from:

- IEEE Conference Templates: https://www.ieee.org/conferences/publishing/templates?authuser=0
- ASYU 2026 Paper Guidelines: https://asyu2026.ozyegin.edu.tr/tr/paper-guidelines?authuser=0

The included IEEE template files are provided only due to University Project Requirements. This project claims no ownership of the IEEE template, formatting materials, or related guideline content.

References should be managed with BibTeX in `references.bib`.
